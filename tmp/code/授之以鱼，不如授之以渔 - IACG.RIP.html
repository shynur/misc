哔咔（PicACG）

关于看本子，目前大多数人知道的地方大概是哔咔了（PicACG）,但最骚的是国内有公司打着哔咔的名头公开宣传公众号/网站，xswl。

话虽这么讲，但假网站+公众号也确确实实的欺骗到了国内不少的人（毕竟懂得科学上网的还是少数）

那么此处便贴示一下

  • 官方网站:https://picacomic.com

  • 哔咔TG公告板：http://t.me/PicACG
  • TG群组似乎因为NSFW原因，无法找到链接。

E-hentai/Exhentai

还有另一个可能你们知道的比较少的网站，E-hentai.国人一般称之为E绅士。

分表站（http://e-hentai.org）和里站（http://exhentai.org）

里站的同人/漫画数量之多堪称本子界的亚历山大图书馆。可以这么说，你在EX-h里面找不到的本子，你在其他地方99%是找不到的。但你在这里能找到的本子，其他地方却可能没有。而本站的绝大部分本子内容也来自于Ex-h。

Ex-h看似挺好，但问题在于，Ex因为涉及的内容偏重口故不符合部分国家法律要求（R-18G/loli等），站长将其进入的门槛略作提高（挡住一大堆人）;不过无伤大雅。

里站准入条件大致如下：

  1. 拥有账号，且注册时间大于一周。
  2. 账号已在表站为登录状态下，才可进入里站。
  3. 无论新老账号，重置密码后一周内均无法进入里站。

具体参照此文章：ExHentai账号简单注册教程

更详细优秀的教程文章:[新版]Ex绅士里站访问教程——注册自己的账号畅游里站

另外可以借助国内某位大佬的作品-EhViewer

当然，这并不是官方客户端，不过鉴于是Github开源，诸位可放心使用。

GitHub Relase地址: https://github.com/seven332/EhViewer/releases/tag/1.7.3

由于原作者已经不再维护此客户端，而近期Eh更改了评论功能，原客户端已无法查阅评论。

但因为源代码是开源的，便有其他大佬Fork了一份接着维护。

Nhentai

官网地址:https://nhentai.net

Nhentai这个站，本质就是EH+EXH的镜像站，甚至创始人之一就是EXH的创始人，两站数据基本上是同步的(有些Eh上面有但是NH没搜到很有可能是名称不一样，别问我怎么知道的

所以也可以前往nhentai去寻找本子。

至于怎么找？复制标题站内搜索会吗崽种？

虽然没有EXH的访问限制，但NH的访问需要梯子辅助，而且我目前为止并没有找到NH的官方/靠谱的第三方客户端，其实它本身移动端适配做的就挺不错，自己摸索一下怎么用吧。

感谢评论区热心老哥的补充，以防万一在此也补上。

关于Nhentai第三方客户端，其实是有的而且体验还不错我列举三个我用的：

1.NHBooks

NH本子很多时候是领先哔咔的，在某年九月开始必须梯子国产，完全符合国人操作习惯，体验良好，缺点是不再更新，但依旧能用最大的毛病是加载时不能返回（会闪退）。

地址：https://github.com/NoireFHC/NHentai-android

2.NClientV2

提供中文语言，不断维护且积极更新（最近一次更新在 22小时前）

地址：https://github.com/Dar9586/NClientV2

3.NHViewer

提供繁体中文（但有待完善），最近一次更新：四天前。

地址：https://github.com/ttdyce/NHentai-NHViewer

此信息来自昵称为『QAQ』的热心老哥，经过验证确实可用。
信息内的时间截止均为2020/6/15

首先来谈谈P站吧，是蓝色的P站，不是黄黑色的

pixiv（日语：ピクシブ）是一个由pixiv股份有限公司运营，主要由日本艺术家所组成的以插图、漫画和小说艺术为中心的社交网络服务里的虚拟社区网站。pixiv于2007年9月10日由上谷隆宏等人首次推出第一个测试版，目前pixiv公司总部位于日本东京都涩谷区千駄谷。[2]pixiv目的是提供一个能让艺术家发表他们的插图，并透过评级系统反应其他用户意见之处，网站以用户投稿的原创图画为中心，辅以标签、书签、作品回应、排行榜等功能形成具有其特色的社交网络。

摘自维基中文

Pixiv被屏蔽也有个几年了，简单一些，你可以透过以下几个第三方项目在大陆直接进行访问。

如何快速寻找插画

首先，你需要一架称手的梯子，可以看第二页:)

当然，如果你可以直接访问当我没说。

其次，下面给出两个以图搜图引擎:

1.acii2d

地址:https://ascii2d.net

大名鼎鼎的插画检索网站，基本上来说只要存在于Pixiv的图像均可在此处查到。

另外，在使用残缺图像搜寻时，如果默认的结果没有匹配到，请点击你所上传图片下方的『特徴検索』按钮，引擎会按照画像的特征重新进行检索，成功率极高（默认的搜寻方式是色合搜索）

2.SauceNAO

地址:https://saucenao.com/

同样是著名的以图搜图引擎，不过相对于ascii2d，它的搜索面会更加广泛一点。但实际上效果却差了那么一点点。我也不知道是什么鬼原因。

此引擎在搜索全图/半图时比较好用，残图就够呛。
